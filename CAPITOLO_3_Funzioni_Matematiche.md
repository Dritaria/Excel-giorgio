# Funzioni Matematiche in Excel

## 1. SOMMA (SUM)
La funzione SOMMA aggiunge insieme un insieme di numeri. È utile per sommare valori numerici in un intervallo.
### Formula:
```
SOMMA(numero1, numero2, ...)
```
### Esempio:
```
=SOMMA(A1:A5)
```
### Caso d'uso:
Può essere utilizzato per calcolare il totale delle vendite in un mese.

## 2. MEDIA (AVERAGE)
La funzione MEDIA restituisce la media aritmetica di un insieme di valori.
### Formula:
```
MEDIA(numero1, numero2, ...)
```
### Esempio:
```
=MEDIA(B1:B5)
```
### Caso d'uso:
Utilizzata per trovare il punteggio medio degli studenti in un esame.

## 3. CONTA (COUNT)
La funzione CONTA restituisce il numero di celle non vuote che contengono numeri.
### Formula:
```
CONTA(valore1, valore2, ...)
```
### Esempio:
```
=CONTA(C1:C10)
```
### Caso d'uso:
Usato per contare il numero di vendite effettuate.

## 4. MINIMO (MIN)
La funzione MINIMO restituisce il valore più piccolo in un insieme di valori.
### Formula:
```
MIN(valore1, valore2, ...)
```
### Esempio:
```
=MIN(D1:D5)
```
### Caso d'uso:
Può essere utilizzato per trovare il prezzo più basso di un articolo in offerta.

## 5. MASSIMO (MAX)
La funzione MASSIMO restituisce il valore più grande in un insieme di valori.
### Formula:
```
MASSIMO(valore1, valore2, ...)
```
### Esempio:
```
=MASSIMO(E1:E5)
```
### Caso d'uso:
Usata per individuare il punteggio massimo in un torneo.

## 6. PRODOTTO (PRODUCT)
La funzione PRODOTTO moltiplica insieme tutti i numeri in un insieme di valori.
### Formula:
```
PRODOTTO(numero1, numero2, ...)
```
### Esempio:
```
=PRODOTTO(F1:F5)
```
### Caso d'uso:
Può calcolare il valore totale di inventario moltiplicando quantità e prezzo per articolo.

## 7. ARROTONDA (ROUND)
La funzione ARROTONDA arrotonda un numero a un numero specificato di cifre decimali.
### Formula:
```
ARROTONDA(numero, num_cifre)
```
### Esempio:
```
=ARROTONDA(G1, 2)
```
### Caso d'uso:
Usato per mostrare valori monetari con un numero fisso di decimali.

## 8. INTERO (INT)
La funzione INTERO restituisce il numero intero più vicino che è minore o uguale a un numero.
### Formula:
```
INTERO(numero)
```
### Esempio:
```
=INTERO(H1)
```
### Caso d'uso:
Usato per calcolare il numero intero della popolazione in un censimento.

## 9. ASSOLUTO (ABS)
La funzione ASSOLUTO restituisce il valore assoluto di un numero, rimuovendo il segno.
### Formula:
```
ASSOLUTO(numero)
```
### Esempio:
```
=ASSOLUTO(I1)
```
### Caso d'uso:
Utilizzato per evitare errori nei calcoli che coinvolgono numeri negativi.

## 10. POTENZA (POWER)
La funzione POTENZA restituisce il risultato di un numero elevato a una certa potenza.
### Formula:
```
POTENZA(numero, potenza)
```
### Esempio:
```
=POTENZA(J1, 2)
```
### Caso d'uso:
Usato per calcolare l'area di un cerchio utilizzando la formula pi * r^2.

## 11. RADICE QUADRATA (SQRT)
La funzione RADICE QUADRATA restituisce la radice quadrata di un numero.
### Formula:
```
RADICE(numero)
```
### Esempio:
```
=RADICE(K1)
```
### Caso d'uso:
Usata per calcolare la lunghezza del lato di un quadrato dato l'area.

## 12. MOD (MOD)
La funzione MOD restituisce il resto della divisione tra due numeri.
### Formula:
```
MOD(numero, divisore)
```
### Esempio:
```
=MOD(L1, 3)
```
### Caso d'uso:
Utile per determinare se un numero è pari o dispari.

---
Queste funzioni sono fondamentali per le analisi matematiche in Excel e possono semplificare notevolmente i calcoli quotidiani.
